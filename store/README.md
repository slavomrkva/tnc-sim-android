# Play Store assets

Marketing/graphics for the Google Play listing (not used by the app build).

| File | Size | Play Console location |
|------|------|-----------------------|
| `icon-512.png` | 512×512 | Store settings → App icon |
| `feature-graphic-1024x500.png` | 1024×500 | Main store listing → Feature graphic |
| `screenshot-1-editor.png` | 1080×1920 | Main store listing → Phone screenshots |
| `screenshot-2-3d.png` | 1080×1920 | Main store listing → Phone screenshots |
| `screenshot-3-learn.png` | 1080×1920 | Main store listing → Phone screenshots |

Screenshots use the app's phone captures (from `www/onboarding/`) with captions.
All are PNG without transparency and within Google Play's required dimensions
(phone screenshots 9:16; feature graphic 1024×500). Upload screenshots in the
order Editor → 3D → Learn.
