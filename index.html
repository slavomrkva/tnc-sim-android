Online CNC simulator compatible with Heidenhain TNC

Free open-source tool for TNC programming and 3D toolpath simulation — right in your browser, no installation needed

Supported commands

  • L X Y Z F — linear move
  • FMAX — rapid positioning
  • CC X Y — arc center
  • C DR+ / DR- — arc move
  • BLK FORM — workpiece
  • TOOL CALL — tool selection

What is TNC

Heidenhain TNC is a CNC control used in milling machines worldwide. This simulator lets you practice TNC programming with a live 3D view — no software to install, just open and start.

How to use

Write TNC code on the left, press Run to animate the tool, or Step to run block by block. Use + / − to change playback speed while it runs. Drag in the 3D view to orbit.

Contribute

This is an early-stage open source project. The source code is available on GitHub.
