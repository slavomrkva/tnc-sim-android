Heidenhain TNC Online Simulator

Free open-source tool for TNC programming and toolpath simulation — directly in your browser, no installation needed

Supported commands

  • L X Y Z F — linear move
  • FMAX — rapid positioning
  • CC X Y — arc center
  • C DR+ / DR- — arc move
  • BLK FORM — workpiece
  • TOOL CALL — tool selection

What is TNC

Heidenhain TNC is a CNC control system used in milling machines worldwide. This simulator lets you practice TNC programming without any software installation — just open and start.

Open source

This project is fully open source. Source code available on GitHub. Contributions welcome — bug reports, new features, translations.
